# 参考网站风格解读

## MotionSites

借鉴点：
- 首屏用超大标题和强视觉主体建立第一印象。
- 将网页拆成可描述的 prompt：背景、字体、主视觉、CTA、滚动方式、响应式。

本网站应用：
- Hero 使用巨大标题、中央头像卡、底部 CTA。
- 简历内容被拆成 About、Internship、Projects、Contact 几个清晰面试叙事模块。

## CSS Design Awards

借鉴点：
- 高级感不只来自特效，而来自留白、节奏、层级、对比和动效克制。
- 适合参考获奖网站的滚动节奏与视觉密度。

本网站应用：
- 使用大面积留白和少量高饱和按钮。
- 动效服务信息展示，不堆砌复杂装饰。

## 21st.dev / React Bits

借鉴点：
- 精细组件：翻转卡片、磁吸按钮、标签、卡片堆叠、动效文字。

本网站应用：
- 自实现工牌翻转、磁吸 Hero 卡、sticky 项目卡、联系页叠放卡片。

## Particleify

借鉴点：
- 个人照片可在后续转换成 3D 粒子版本，适合作为 Hero 的 hover 备选视觉或彩蛋。

本网站应用：
- 当前先用照片占位，避免没有真人图片时产生空效果。
- 等你提供照片后，可以把粒子版本作为第二视觉素材加入。

## Anime.js 与 GSAP

借鉴点：
- Anime.js 适合轻量局部动画。
- GSAP 更适合滚动叙事、时间线和性能稳定的页面动效。

本网站应用：
- 当前主要采用 GSAP：入场、滚动、磁吸、文字透明度、横向滚动和项目卡缩放。
